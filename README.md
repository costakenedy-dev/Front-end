## 🚀 Como Usar Localmente

1. **Clone ou baixe o repositório**
   ```bash
   git clone <seu-repositorio>
   cd sports-landing-page
   ```

2. **Abra no navegador**
   - Abra o arquivo `index.html` diretamente no navegador, ou
   - Use um servidor local:
   ```bash
   python3 -m http.server 8000
   ```
   Então acesse `http://localhost:8000`

## 🌐 Deploy no GitHub Pages

### Passo 1: Crie um repositório no GitHub
1. Acesse [github.com](https://github.com) e faça login
2. Clique em "New repository"
3. Nomeie como `sports-landing-page`
4. Deixe público e clique em "Create repository"

### Passo 2: Configure o GitHub Pages
1. Vá para as configurações do repositório (Settings)
2. No menu lateral, clique em "Pages"
3. Em "Source", selecione a branch `main` (ou `master`)
4. Em "Folder", selecione `/docs`
5. Clique em "Save"

### Passo 3: Faça o upload dos arquivos
1. Extraia o arquivo ZIP
2. Copie todos os arquivos para seu repositório local
3. Execute os comandos:
   ```bash
   git add .
   git commit -m "Initial commit - SportHub landing page"
   git push origin main
   ```

### Passo 4: Acesse sua página
Após alguns minutos, sua landing page estará disponível em:
```
https://seu-usuario.github.io/sports-landing-page/
```
